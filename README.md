# What to Wear (WTWR)

This is a website that suggests what clothes you should wear based on the current weather of user. The weather is fetched from [OpenWeather API](https://openweathermap.org/api/one-call-3#example).

## Technologies Used:

- React.js
- Vite
- OpenWeather API
